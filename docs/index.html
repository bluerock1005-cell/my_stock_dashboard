my_stock_dashboard

一个基于 Python 的 A 股实时行情监控与告警工具，支持命令行与 PyQt6 图形界面双模式展示。

Python A股 实时监控 PyQt6

功能亮点

  • 实时拉取个股和 ETF 行情数据
  • 支持价格阈值告警（买入价 / 卖出价）
  • 支持自定义监控列表及买卖数量
  • 包括命令行和 GUI 两种使用体验
  • 可打包为独立 Windows 可执行文件

快速上手

克隆仓库后，安装依赖并运行：

git clone https://github.com/bluerock1005-cell/my_stock_dashboard.git
cd my_stock_dashboard
pip install -r requirements.txt

命令行模式

python stock_monitor.py

图形界面模式

python gui.py

配置说明

项目使用 watchlist.json 存储监控股票和阈值配置，支持两种格式：

  • 仅代码数组
  • 包含 upper, lower, sell_qty, buy_qty, note 的对象数组

源码说明

  • stock_monitor.py：命令行入口
  • gui.py：PyQt6 GUI 界面入口
  • fetcher.py：行情抓取逻辑
  • alerter.py：价格告警判断
  • display.py：命令行渲染表格
  • formatter.py：数据格式化工具
  • config.py：基础配置项
